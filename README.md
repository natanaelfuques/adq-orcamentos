# Áudio de Qualidade — Sistema de Gestão

Sistema web interno para gestão de eventos, contratos, orçamentos, estoque e financeiro da empresa Áudio de Qualidade.

## Páginas

| Arquivo | Função |
|---|---|
| `index.html` | App principal — orçamentos, contratos e eventos rápidos |
| `painel_eventos.html` | Kanban de eventos confirmados |
| `relatorios.html` | Relatórios financeiros e controle de pagamentos |
| `estoque.html` | Controle de estoque de equipamentos |
| `orcamento-view.html` | Visualização pública de orçamentos (link para cliente) |
